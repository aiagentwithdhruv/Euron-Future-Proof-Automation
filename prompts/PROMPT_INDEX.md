# Prompt Library Index

> All reusable prompts organized by category. Update this file whenever you add a new prompt.

---

## How to Add a Prompt

1. Copy `templates/PROMPT_TEMPLATE.md` to the appropriate category folder
2. Rename it descriptively (e.g., `fastapi-crud-generator.md`)
3. Fill in all fields — name, tags, the prompt, variables, examples
4. Add an entry to this index below
