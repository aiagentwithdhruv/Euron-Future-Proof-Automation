## What Is Angelina?

Angelina is a 5-agent AI orchestrator with 30+ tools, Gemini Live voice interaction, and pgvector memory. She's not a chatbot — she's a full AI assistant that remembers context, takes actions, and gets things done across your entire digital life.

**Agents:**
1. Research Agent — deep web research, fact-checking, competitive analysis
2. Content Agent — writes posts, emails, docs in your voice
3. Action Agent — books meetings, sends emails, manages tasks
4. Memory Agent — remembers everything, surfaces context when needed
5. Orchestrator — routes requests to the right agent, manages multi-step workflows
